IMPORTANT: Additional Package Added
=====================================

A new visualization library has been added for creating business framework diagrams.

WHAT WAS ADDED:
- Mermaid.js for creating SWOT, Porter's 5 Forces, BCG Matrix, and other framework visualizations

TO INSTALL:
Run this command to update your dependencies:

    npm install

This will install the mermaid package and enable visual diagrams in the PPT Frameworks section.

FEATURES:
The PPT Frameworks page now includes automatic visual diagrams for:
✓ SWOT Analysis
✓ Porter's 5 Forces
✓ BCG Matrix
✓ PESTLE Analysis
✓ Value Chain
✓ 7Ps Marketing Mix
✓ Ansoff Matrix
✓ TAM/SAM/SOM

After running npm install, restart your dev server:
    npm run dev
